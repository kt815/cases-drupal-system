
Installation
---------------------

 * git clone https://github.com/kt815/cases-drupal-system.git .

 * Unpack db/db_cases.sql in your database

 * Change sites/default/settings.php (choose your own):
      'database' => 'kt815_d7',
      'username' => 'root',
      'password' => '1',

 * Login: http://localhost/user/
